## ğŸš€ About This Project

This repository contains the standalone **Official Menu Page** for the "IGNITE 2025" event. Its primary purpose is to showcase the delicious Pan-Asian cuisine provided by the event's venue partner, **TOQUO**, in an engaging and easy-to-navigate format.

Built purely with **HTML**, **Tailwind CSS** (via CDN), and minimal **Vanilla JavaScript** for interactivity, this page is designed to be lightweight, responsive, and visually consistent with the main IGNITE 2025 branding. It uses an accordion layout to neatly categorize menu items.

---

## âœ¨ Key Features

* **Clear Menu Presentation:** Displays starters, main courses, and desserts in distinct, collapsible sections using an accordion interface.
* **Visual Appeal:** Features high-quality images and detailed descriptions for each menu item.
* **Venue Spotlight:** Includes information about the host venue, **TOQUO**, enhancing the event's context.
* **Responsive Design:** Adapts seamlessly to various screen sizes (desktops, tablets, mobiles) thanks to **Tailwind CSS**.
* **Interactive Accordion:** Uses simple Vanilla JavaScript to create a smooth expand/collapse effect for menu sections.
* **Themed Styling:** Matches the dark, neon-green aesthetic of the main IGNITE 2025 event branding.
* **Standalone & Lightweight:** No complex dependencies or build steps required; runs directly in the browser.
